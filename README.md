# Storage-Service

Сервис для безопасного обмена данными между двумя пользователями сайта.

Для асимметричного шифрования используются следующие алгоритмы:

- RSA-4096 (обмен ключами и шифрование ключей)
- Curve25519 (ECDH для согласования ключей)

// AES-256-GCM (симметричное шифрование данных)

Каждый пользователь может создать себе ключи (приватные и публичные) с ограниченным сроком годности и доверяет сайту их хранение.

Данные передаются через форму.
Можно вбить их напрямую в поле ввода или выбрать файл.

Сервис исключает промежуточное хранение данных в незашифрованном виде.

Позволяет хранить зашифрованные данные любого объема в облаке где угодно.

Алгоритмы шифрования пользователь может настроить для себя.
